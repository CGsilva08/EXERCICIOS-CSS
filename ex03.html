Exercicio 03-css externo

Note que nesse caso não existe nenhum código CSS na página HTML, pois o arquivo que contém as formatações é externo. Para isso você deve criar um arquivo .css em uma pasta "Style" e indicá-lo através de um link dentro HEAD.

Neste caso todos os parágrafos irão herdar as características definidas pelo arquivo .css, note que qualquer arquivo HTML que você criar e indicar o arquivo css externo será formato pelo padrão definido, isso é muito útil para projetos com mais de uma arquivo HTML.
